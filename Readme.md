# **U-Turn**

<p align="center"> <img src="Assets/Art/ReadMe UI/readme_img.png" alt="Game Instructions"> </p>

自動で加速する車を操作し、スペースキーだけでスピードを調整しながら難しいカーブを走り抜ける3Dレースゲーム。
手描き風のスタイルと物理ベースの動きを特徴としています。

---

## 🎮 **ゲーム概要**

* 車は **自動で前進・加速**
* プレイヤーは **スペースキーで加速／減速のみ操作**
* 速度が高すぎるとカーブで **コースアウトしてゲームオーバー**
* 制限時間内に **規定の周回数を完走** するとクリア

---

## 🕹 **操作**

| アクション | キー        |
| ----- | --------- |
| 加速・減速 | **Space** |
| 進む    | **Enter** |

---

## 📄 **画面内説明文（レベル1）**

* スペースキーで加速／減速。速度が高すぎるとカーブでコースアウトしてゲームオーバー。
* 制限時間内に規定の周回数を走り切れば勝利。
* Enterキーを押して続ける。

---

## 🛠 **特徴**

* Rigidbody＋AddForce を使った物理ベースの車挙動
* スプラインの角度と速度によりカーブ難易度が変化
* 手描き風のアート、動く雲影（ライトのクッキー＋CRT）
